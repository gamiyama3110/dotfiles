## Installation

初めてこの設定を適用する場合。
環境のdotfilesに上書きするので注意。

```
git clone https://github.com/gamiyama3110/dotfiles.git && cd dotfiles && source bootstrap.sh
```

更新する場合はdotfilesのディレクトリに移動して`bootstrap.sh`を実行。

```
set -- -f; source bootstrap.sh
```
